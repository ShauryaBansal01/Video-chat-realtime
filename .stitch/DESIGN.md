# Streamify Design System

## Brand Mood
- Warm editorial
- Globally connected
- Conversation-first
- Romanticized language learning, not generic productivity software

## Themes
- `streamify_daybreak`: bright parchment surfaces, terracotta actions, deep ink text, sage secondary accents
- `streamify_midnight`: espresso surfaces, warm clay highlights, sea-glass secondary accents, candlelight text
- `streamify_olive`: linen surfaces, olive secondary accents, saffron warmth
- `streamify_tide`: coastal paper surfaces, teal depth, sea-glass softness
- `streamify_ember`: dark plum-espresso surfaces with copper energy
- `streamify_bloom`: soft blush paper with berry and botanical accents

## Typography
- Display: `Fraunces`
  - Use for hero headlines, brand mark, and key section titles
- Body/UI: `Manrope`
  - Use for navigation, forms, cards, metadata, and dense content

## Spacing
- Tight UI spacing should still feel breathable
- Standard vertical rhythm favors `gap-4`, `gap-6`, `gap-8`
- Page sections should feel editorial, using generous top spacing and distinct content bands

## Surfaces
- Primary surfaces are softly elevated cards with warm borders and tinted backgrounds
- Radius is consistently generous: pills for controls, rounded cards for content blocks
- Shadows are soft and warm, never cold or harsh

## Motion
- Use subtle reveal motion on section load
- Hover states should lift slightly and increase contrast
- Motion should reinforce warmth and polish, not playfulness

## Components
- Navigation should feel like a branded rail, not an admin sidebar
- Empty states should be elegant and optimistic
- Chat and call UI should inherit Streamify shell styling even when underlying Stream components remain intact
